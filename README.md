# Ejercicio de Gestión de Asignaturas

Este proyecto es un ejemplo de una aplicación web que permite gestionar asignaturas en una base de datos. Los usuarios pueden realizar las siguientes operaciones:

- Listar todas las asignaturas almacenadas en la base de datos.
- Crear una nueva asignatura.
- Actualizar los datos de una asignatura existente.
- Obtener información detallada de una asignatura por su ID.
- Borrar una asignatura por su ID.

## Tecnologías Utilizadas

Este proyecto utiliza las siguientes tecnologías:

- Java: Para la lógica del servidor.
- Servlets: Para manejar las solicitudes HTTP.
- MySQL: Como sistema de gestión de base de datos.
- HTML: Para la interfaz web básica.

## Configuración

Antes de ejecutar la aplicación, asegúrate de tener lo siguiente configurado:

- Un servidor web compatible con Servlets (como Tomcat) configurado en tu entorno de desarrollo.
- Una base de datos MySQL configurada con el nombre `mi_bbdd_proyecto`.
- Asegúrate de que el controlador JDBC de MySQL está en el classpath del proyecto.

## Ejecución

1. Clona este repositorio en tu sistema local.
2. Importa el proyecto en tu entorno de desarrollo (por ejemplo, Eclipse o IntelliJ IDEA).
3. Asegúrate de que tu servidor web (Tomcat u otro) esté configurado y en funcionamiento.
4. Ejecuta la aplicación desde tu entorno de desarrollo.
